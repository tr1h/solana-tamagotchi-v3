⚡ БЫСТРЫЙ СТАРТ (5 МИНУТ):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. УСТАНОВИТЕ PHANTOM WALLET
   → Перейдите: https://phantom.app/
   → Установите расширение для браузера
   → Создайте новый кошелек (сохраните seed фразу!)
   → Переключитесь на DEVNET (Настройки → Сеть → Devnet)

2. ПОЛУЧИТЕ БЕСПЛАТНЫЙ SOL (ДЛЯ ТЕСТА)
   → Перейдите: https://solfaucet.com/
   → Вставьте адрес кошелька
   → Получите 1-2 SOL

3. ЗАПУСТИТЕ ЛОКАЛЬНЫЙ СЕРВЕР
   
   Выберите ОДИН способ:
   
   🐍 Python (обычно уже установлен):
      python -m http.server 8000
   
   📦 Node.js:
      npx http-server -p 8000
   
   🐘 PHP:
      php -S localhost:8000

4. ОТКРОЙТЕ В БРАУЗЕРЕ
   → http://localhost:8000

5. ИГРАЙТЕ!
   → Подключите кошелек
   → Создайте питомца (0.1 SOL на Devnet)
   → Начинайте играть!

🎮 ИГРОВАЯ МЕХАНИКА:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Характеристики (0-100):
  🍔 Голод      → Уменьшается на 2/мин, пополняется едой
  ⚡ Энергия    → Уменьшается на 1/мин, пополняется сном
  😊 Счастье    → Уменьшается на 1/мин, пополняется игрой
  ❤️ Здоровье   → Уменьшается при голод/счастье < 20

Действия:
  🍔 Покормить  → +25 голод, -5 энергия, +10 XP
  🎮 Играть     → +20 счастье, -15 энергия, -10 голод, +15 XP
  😴 Спать      → +30 энергия, -5 счастье, +5 XP
  💊 Лечить     → Полное здоровье (10 TAMA или 0.01 SOL)

Прогрессия:
  📈 Повышение уровня → Зарабатывайте XP (XP = 100 × 1.5^уровень)
  ✨ Эволюция → Каждые 10 уровней (50 TAMA)
  🏆 Достижения → Выполняйте задания за награды

Экономика:
  💰 Создать питомца → 0.1 SOL
  💀 Воскресить → 0.05 SOL или 100 TAMA
  💊 Вылечить → 10 TAMA
  ✨ Эволюция → 50 TAMA
  🎁 Дневная награда → 50 TAMA (каждые 24ч)
  👥 Реферал → 25 TAMA за друга

🔧 НАСТРОЙКА (ПЕРЕД ДЕПЛОЕМ):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Firebase (Опционально - для облачных функций):
   → Создайте проект: https://console.firebase.google.com/
   → Включите Firestore Database
   → Скопируйте конфиг в js/database.js
   → См. SETUP.md для деталей

2. Смена сети (для продакшена):
   → Отредактируйте js/wallet.js
   → Измените: network: 'devnet' → 'mainnet-beta'

3. Кошелек казначейства (ВАЖНО!):
   → Отредактируйте js/wallet.js
   → Замените: treasuryWallet = 'ВАШ_АДРЕС_КОШЕЛЬКА'

4. Иконки (Опционально):
   → Замените placeholder иконки в manifest.json
   → Используйте https://realfavicongenerator.net/

Проблема: Кошелек не подключается
Решение:
  → Проверьте, что Phantom установлен
  → Обновите страницу (Ctrl+F5)
  → Очистите кэш браузера
  → Попробуйте другой браузер

Проблема: Транзакция не проходит
Решение:
  → Проверьте баланс SOL
  → Убедитесь в правильной сети (Devnet/Mainnet)
  → Проверьте что Phantom разблокирован
  → Смотрите ошибки в консоли (F12)

Проблема: Игра не загружается
Решение:
  → Проверьте что все файлы на месте
  → Откройте консоль (F12) для ошибок
  → Очистите кэш браузера
  → Убедитесь что сервер запущен

Проблема: Ошибки Firebase
Решение:
  → Проверьте конфиг в js/database.js
  → Убедитесь что Firestore включен
  → Проверьте правила безопасности
  → Смотрите консоль браузера

   → Бесплатный SOL из faucet
   → Нет реальной стоимости
   → Безопасно для тестов
   → Используйте сначала это!

🔴 MAINNET (Продакшн):
   → Нужен реальный SOL
   → Реальные деньги
   → Сначала протестируйте
   → Используйте осторожно

🔒 Безопасность:
   → НИКОГДА не делитесь seed фразой
   → Делайте резервные копии кошелька
   → Сначала тестируйте на Devnet
   → Измените адрес кошелька казначейства
   → Используйте свой Firebase конфиг
