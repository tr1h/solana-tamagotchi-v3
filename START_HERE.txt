📋 QUICK START (5 MINUTES):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. INSTALL PHANTOM WALLET
   → Visit: https://phantom.app/
   → Install browser extension
   → Create new wallet
   → Switch to DEVNET (Settings → Change Network → Devnet)

2. GET FREE SOL (FOR TESTING)
   → Visit: https://solfaucet.com/
   → Paste your wallet address
   → Get 1-2 SOL instantly

3. START LOCAL SERVER
   
   Choose ONE method:
   
   🐍 Python (usually pre-installed):
      python -m http.server 8000
   
   📦 Node.js:
      npx http-server -p 8000
   
   🐘 PHP:
      php -S localhost:8000

4. OPEN IN BROWSER
   → http://localhost:8000

5. PLAY!
   → Connect Wallet
   → Create Pet (0.1 SOL on Devnet)
   → Start playing!

🎮 GAME MECHANICS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Stats (0-100):
  🍔 Hunger   → Decreases 2/min, restore by feeding
  ⚡ Energy   → Decreases 1/min, restore by sleeping
  😊 Happiness → Decreases 1/min, restore by playing
  ❤️ Health   → Decreases when hunger/happy < 20

Actions:
  🍔 Feed    → +25 hunger, -5 energy, +10 XP
  🎮 Play    → +20 happy, -15 energy, -10 hunger, +15 XP
  😴 Sleep   → +30 energy, -5 happy, +5 XP
  💊 Heal    → Full health (costs 10 TAMA or 0.01 SOL)

Progression:
  📈 Level Up → Earn XP to level up (XP = 100 × 1.5^level)
  ✨ Evolution → Every 10 levels (costs 50 TAMA)
  🏆 Achievements → Complete challenges for rewards

Economy:
  💰 Create Pet → 0.1 SOL
  💀 Revive Pet → 0.05 SOL or 100 TAMA
  💊 Heal Pet → 10 TAMA
  ✨ Evolve Pet → 50 TAMA
  🎁 Daily Reward → 50 TAMA (every 24h)
  👥 Referral → 25 TAMA per friend

🐛 TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: Wallet won't connect
Solution:
  → Check Phantom is installed
  → Refresh page (Ctrl+F5)
  → Clear cache
  → Try different browser

Problem: Transaction fails
Solution:
  → Check you have enough SOL
  → Verify correct network (Devnet/Mainnet)
  → Check Phantom is unlocked
  → Review console errors (F12)

Problem: Game doesn't load
Solution:
  → Check all files are present
  → Open console (F12) for errors
  → Clear browser cache
  → Verify server is running

Problem: Firebase errors
Solution:
  → Verify config in js/database.js
  → Check Firestore is enabled
  → Review security rules
  → Check browser console
